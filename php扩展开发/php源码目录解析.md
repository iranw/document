
# php源码目录结构解析

```
源码根目录/
    |---README.PHP4-TO-PHP5-THIN-CHANGES            #列举了PHP4和PHP5的一些差异
    |---CODING_STANDARDS                            #代码规范
    |---build                                       #编译相关文件
    |---ext                                         #官方扩展目录
    |---main                                        #php核心基础文件
    |---Zend                                        #Zend引擎实现代码
    |---pear                                        #php扩展与应用仓库
    |---sapi                                        #服务器抽象层代码
    |---TSRM                                        #tsrm库 线程安全处理库
    |---tests                                       #php测试脚本目录
    |---win32                                       #windows平台实现
    |---run-tests.php                               #测试脚本入口
```
